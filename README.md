<h1>What is the BirdHub?</h1>
The BirdHub is a collection of online games ported from Github. It is designed to be used by students on their school devices, allowing them to play online games even under common device security systems.
The BirdHub is owned by web developer AnonymousBirb5100. The BirdHub is frequently updated with new games and/or new web features.
All of the BirdHub and its games are programmed in HTML and JavaScript.
